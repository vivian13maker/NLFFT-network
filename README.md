# README

This repository implements NLFFTNet. The implementation is heavily influenced by the projects RFB, Detr. This repository aims to be the code base for researches based on RFB, Detr.

## Installation 

1. Python3
1. PyTorch 1.5+ or higher
1. yacs
1. tqdm
1. scipy
1. GCC >= 4.9
1. OpenCV
